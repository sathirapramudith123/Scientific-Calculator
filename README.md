Project Title <b>Scientific Calculator</b>

Description
A C++-based scientific calculator application with support for advanced mathematical operations.

Features

Basic arithmetic: +, -, *, /<br>
Scientific functions: trigonometric (sin, cos, tan), logarithms, powers<br>
Possibly user-input interface via command line or UI

instalation
```
git clone https://github.com/sathirapramudith123/Scientific-Calculator.git
cd Scientific-Calculator
```
